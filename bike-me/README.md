## Bike Me

You will be using the [City Bikes API](http://api.citybik.es/v2/).

> You do not need an api token for this exercise.

#### Challenge
* How many bikes are available at the following *Bay Area Bike Share* locations:
    * Montgomery
    * Powell
    * Civic Center
